üìã How to Embed the Widget

To add this widget to your website, simply copy and paste this code before the closing </body> tag:

<!-- IT Helpdesk Widget --> <script> window.HelpdeskWidget = { apiUrl: 'http://localhost:5000', // Your Flask app URL theme: 'dark', // 'light' or 'dark' position: 'bottom-right', // 'bottom-right' or 'bottom-left' greeting: 'Hi! How can I help you today?', allowUploads: true, allowUrls: true, maxFileSize: 10, primaryColor: '#667eea', secondaryColor: '#764ba2' }; </script> <script src="http://localhost:5000/static/widget.js"></script> <!-- End Widget -->
